This page provides the following temperature data:

  1. Provides an overview of temperature on earth.
  2. Provides an overview of temperature by hemisphere.
  3. Provides some statistical data associated with the hemisphere plots.
*Conclusion*: The global temperature plot shows that the farther away from the equator (Latitude = 0), the max temp is colder (lower temperatures). The linear regression model for the northern hemisphere shows that the farther south you go in the northern hemisphere, the temperature increases. The data is clustered around the line showing a strong correlation. The linear regression model for the southern hemisphere shows that the temperature decreases in the southern hemisphere the farther away you get from the equator (lat = 0). There seems to be a strong correlation as the cluster is consistently, but not entirely, around the regression line.
