Welcome to my website! The purpose of this website is to:

  1. Provide statistical data associated with the global climate.
  2. Ilustrate global temperature, humidity, cloudiness, and windspeed.
  3. Provide a view of comparisons for the northern and southern hemisphere.
  4. Share data that was collected through an open API (see GitHub repository link).
This website may support a variety of needs such as identifying ideal vacation destinations based on personal references, educating ourselves on weather patterns, etc.
